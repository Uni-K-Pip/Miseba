Miseba — 配信の見どころを、チャットから発掘。

Miseba は、YouTube ライブ配信のアーカイブから「切り抜き動画」を作る編集者向けの Chrome 拡張機能です。視聴ページ上で見どころに打点し、盛り上がりを解析して、編集ソフトへそのまま渡せる形で書き出します。

主な機能

無料

  • 見どころの打点（イン点・アウト点・メモ）
  • YouTube チャプター用テキストの書き出し

Pro（月額）

  • チャット盛り上がり解析・ヒートマップ表示
  • チャット検索ジャンプ
  • CSV / EDL / FCPXML 形式での書き出し（編集ソフト連携）

料金

月額 980 円（Pro プラン）。自動更新のサブスクリプションで、いつでも解約できます。解約は拡張機能のパネルまたはポップアップの「サブスクリプションを管理」から行えます。無料機能はサブスクリプションなしで利用できます。

入手・ご利用方法

  • Chrome ウェブストアから Miseba をインストールします。
  • YouTube の視聴ページ（https://www.youtube.com/watch*）を開くと、画面上にパネルが表示されます。
  • Pro 機能を使う場合は、パネルまたはポップアップの「アップグレード」から月額プランに申し込みます。

お支払い・課金について

Pro の決済は、決済代行サービス ExtensionPay（Stripe 決済）を通じて行われる月額自動更新のサブスクリプションです。本拡張は自前の決済サーバーを持たず、カード番号等の支払い情報を保持・取得しません。解約後は、その課金期間の満了をもって自動更新が停止します。

プライバシーポリシー

最終更新: 2026-06-28

概要

Miseba（以下「本拡張」）は、利用者のプライバシーを最優先に設計されています。本拡張は、利用者の閲覧内容や打点・設定データを収集・送信・共有しません。唯一の例外は、有料機能（Pro）の購入手続きで、決済代行サービス ExtensionPay / Stripe にメールアドレス等の決済情報が送られる場合のみです（後述）。

取得・保存する情報

本拡張が端末内に保存するのは、利用者がブラウザ上で作成・設定した次の情報のみです。

  • 打点情報（イン点・アウト点の時刻、メモ、ひも付ける動画 ID）— IndexedDB
  • 設定（書き出し fps、チャット検索のカスタム辞書）— chrome.storage
  • 課金状態のキャッシュ（Pro か否か）— ExtensionPay が chrome.storage に保存

これらは利用者のブラウザ内にローカル保存され、利用者の端末から外部へ送信されることはありません（課金状態の照会を除く）。本拡張は独自の外部サーバーを持ちません。

チャットの解析について

盛り上がりヒートマップを作るために、本拡張は再生中の動画の公開チャットリプレイを YouTube から読み取り、件数や時間あたりの密度を利用者の端末内で集計します。チャットの本文を外部に送信・永続保存・再配布することはありません。集計結果（ヒートマップ）はメモリ上で扱われ、保存されるのは利用者が作成した打点のみです。

取得しない情報

  • 閲覧履歴・検索履歴
  • 位置情報・連絡先
  • カード番号等の支払い情報（決済は ExtensionPay / Stripe が処理し、本拡張はカード情報を保持・取得しません）

本拡張はアクセス解析・トラッキング・広告の仕組みを一切含みません。

課金（Pro）について

有料機能（Pro）の決済には、決済代行サービス ExtensionPay（Stripe 決済）を利用します。本拡張は自前の決済サーバーを持ちません。

  • 購入手続きの際、ユーザーのメールアドレス・決済情報は ExtensionPay / Stripe へ送られ、これらのサービスのプライバシーポリシーに基づいて取り扱われます。
  • 本拡張がそれらのサービスから受け取り保持するのは「課金済みか否か」の状態のみで、カード番号等の支払い情報は受け取りません。
  • 決済完了の検知（onPaid）のため、https://extensionpay.com/* 上でメッセージの受け渡しを行います。

動画について

本拡張は再生位置の移動やヒートマップの表示を行うだけで、動画や音声・チャットのダウンロード・保存・再配布は行いません。

権限

本拡張の主機能は YouTube の視聴ページ（https://www.youtube.com/watch*）でのみ動作します。チャットの取得も同一オリジン（youtube.com）への通信のみです。加えて、Pro 決済の状態同期のために https://extensionpay.com/* でコンテンツスクリプトが動作し、バックグラウンドの service worker が課金状態を管理します。これは ExtensionPay による決済完了の検知と課金状態の取得のためのみに使用し、それ以外のページやデータにはアクセスしません。

データの削除

保存された打点は、本拡張のパネルから削除できるほか、ブラウザのサイトデータ（YouTube）を消去するか、拡張機能を削除することでいつでも削除できます。

変更

本ポリシーを更新する場合は、本ページの「最終更新」を改訂します。
